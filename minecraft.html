Minecraft

Join the EIU Minecraft server!

This page is a work in progress, but join the EIU Minecraft server! You can join on any device - a PC running Java or Bedrock, your phone, or your console (PS, Xbox, Switch)!

The server IP is mc.eiucs.club

The server is shared between members of the EIU Computer Science Club, the EIU eSports Club, and the EIU Gamer's Guild. Membership in these clubs isn't required to join the server, but it can make it more fun!
