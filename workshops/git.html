Git Workshop

  1. Download and open 64-bit Git Portable (https://git-scm.com/download/win)
  2. Open the PortableGit folder in your Downloads folder and open git-bash.exe
  3. Run the following commands:
    • git config --global user.name “YourNameHere”
    • git config --global user.email “YourEmailHere”
    • mkdir folder
  4. Go to gitea.eiucs.club
  5. Register an account (this is not publicly accessible, nor will it stay around after this workshop, so feel free to type in a fake email and an easy password)
  6. Once you’ve signed in, go to the Explore tab and find “will / git-workshop”
  7. Fork the repository (button on the top-right side of the repository)
  8. Enter the following commands
    • git clone https://gitea.eiucs.club/yourusername/git-workshop.git ./folder
    • git init
    • cd folder
    • cd coolpeople
    • echo Your Name > yourname.txt
    • git add .
    • git commit -m “Added Your Name to coolpeople”
    • git push -u origin main
  9. A prompt should come up, select the UI option and enter your credentials for Gitea
  10. You should be able to see your changes by directing your browser to https://gitea.eiucs.club/yourusername/git-workshop
  11. Go back to https://gitea.eiucs.club/will/git-workshop
  12. Go to the pull requests tab
  13. Create a new pull request
  14. Your request should be merge into: will:main … pull from: username:main
  15. Finish creating the pull request
  16. Will can now merge your request
